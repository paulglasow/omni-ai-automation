Most people use one AI at a time. OmniAI v4 lets you use 4 AIs as a team:

```

  Result better than any single AI
```

## Section 1: Creative Writing

### Short Stories & Fiction

**Step 1 — Research phase (Perplexity)**
```
Research phase: I'm writing a story set in [time/place].
Find me 5 surprising true facts I can weave into the story 
to make it feel authentic and interesting.
```

**Step 2 — Structure phase (Gemini)**
```
Structure phase: Using these facts: [paste from step 1]
Create a 5-act story structure for a [genre] story about [premise].
Include: setup, conflict, rising action, climax, resolution.
```

**Step 3 — Writing phase (ChatGPT)**
```
Writing phase: Using this structure: [paste from step 2]
Write the full story in [word count] words. 
Use vivid sensory details. Show, don't tell.
Make the opening line unforgettable.
```

**Step 4 — Editing phase (Claude)**
```
Editing phase: Review this story: [paste from step 3]
Improve: pacing, character voice, emotional resonance, and clarity.
Flag any logic inconsistencies. 
Provide the edited version with changes explained.
```

### Poetry

```
Write a [type: sonnet/haiku/free verse/ballad] about [theme].

Using all 4 AI perspectives:
- ChatGPT: Write a version with bold imagery and wordplay
- Claude: Write a version focused on emotional depth and precision
- Gemini: Provide the technical analysis (meter, rhyme scheme, devices used)
- Perplexity: Find 2 famous poems on this theme I can learn from

Give me all 4 outputs, then synthesize the best version.
```

### Screenwriting / Scripts

```
I want to write a [short film/podcast episode/YouTube script] about [topic].

Phase 1 - Concept (Perplexity): What's trending in [genre] right now?
Phase 2 - Structure (Gemini): Create a 3-act structure with scene-by-scene outline
Phase 3 - Dialogue (ChatGPT): Write the opening scene with natural, snappy dialogue
Phase 4 - Polish (Claude): Review for pacing, subtext, and character consistency
```

## Section 2: Music

### Songwriting

**Lyric writing workflow:**

```
I'm writing a [genre] song about [theme]. Artist influences: [artists].

Step 1 - Hook (ChatGPT): Write 5 different chorus hook options
Step 2 - Story (Claude): Develop the narrative arc for verses 1 and 2
Step 3 - Bridge (Gemini): Analyze the chord progressions in songs by [influence]
         and suggest a bridge that creates tension
Step 4 - Full song (ChatGPT): Assemble the best elements into a complete song
Step 5 - Review (Claude): Does the story arc work? What's missing emotionally?
```

## Section 4: Learning New Skills

### The 30-Day Learning Sprint

```
I want to learn [skill] in 30 days. I'm a [beginner/intermediate].
I have [X hours per week] available.

Using all 4 AIs, create:

Perplexity: What are the best current learning resources (books, courses, YouTube)?

ChatGPT: Make the learning plan engaging with mini-projects and games
Claude: Add accountability checkpoints, progress tests, and a final project

Output: A complete 30-day learning plan I can follow day by day.
```

## Section 6: Research & Deep Dives

### The 4-Perspective Research Method

For any topic you want to deeply understand:

```
I want to deeply understand [topic].

Round 1 — Foundation:
- Perplexity: What's the current state of knowledge? Latest research?
- Gemini: Key statistics, data, and historical timeline
- Claude: What are the biggest debates and open questions?
- ChatGPT: Explain it to me like I'm a curious 15-year-old

Round 2 — Deep dive:
Based on Round 1, what's the most interesting or surprising aspect?
Go deeper on that with all 4 AIs.

Round 3 — Application:
How does this knowledge apply to my life/work/interests?
What should I do with what I've learned?
```

---

### Book Deep Dives

```
I just read [book title] by [author].

Help me get more from it:
- Claude: Analyze the key arguments and their logical structure
- ChatGPT: Find 5 interesting ways to apply the ideas in daily life
- Gemini: Compare this book's ideas to related research and other books
- Perplexity: What do critics and experts say about this book's impact?

Then: What should I read next that builds on these ideas?
```

## Section 7: Prompt Templates Library

Save these for quick access:

### Daily creativity warm-up
```
Give me a creative challenge for today — something I can do in 30 minutes 
related to [your hobby]. Make it specific and slightly outside my comfort zone.
```

### Brainstorming session
```
I'm stuck on [problem]. Give me 20 completely different ideas — 
from obvious to wild. No filtering. Quantity over quality.
```

### Critique request
```
Here's my [work: writing/design/plan]: [paste it]
Give me honest feedback. Don't be gentle. What's not working and why?
Then tell me exactly how to fix the top 3 issues.
```

### "Teach me by example"
```
I want to understand [concept] by looking at 3 excellent examples of it.
For each: explain what makes it excellent, and what I can steal from it.
```

### The "Yes, and" technique
```
My idea: [describe idea]
Build on it using the improv "yes, and" technique — 
accept everything and add to it until we reach something unexpectedly great.
```
